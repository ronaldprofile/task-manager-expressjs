# Documentação do Projeto

Bem-vindo à documentação do Task Manager API.

## Estrutura

Esta pasta contém toda a documentação técnica e de produto do projeto:

- **[PRDs](./prd/)** - Product Requirements Documents detalhando as features do sistema
- **README.md** (este arquivo) - Visão geral da documentação

## Product Requirements Documents (PRDs)

Os PRDs descrevem em detalhes as funcionalidades do sistema, incluindo:

- Requisitos funcionais e não funcionais
- Especificações técnicas
- Modelos de dados
- Critérios de aceitação

### PRDs Disponíveis

- [Authentication & Authorization](./prd/authentication-authorization.md)
- [Tasks Management](./prd/tasks-management.md)
- [Team Management](./prd/team-management.md)

## Como Contribuir

Ao criar novas features:

1. Crie um PRD na pasta `prd/`
2. Atualize o índice em `prd/README.md`
3. Mantenha a documentação atualizada conforme a implementação evolui
